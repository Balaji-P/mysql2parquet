# MySQL to parquet
Converts the output of a MySQL query to parquet

## Usage mysql2parquet
```
Usage of ./mysql2parquet:
  -connectionString string
        MySQL connectionstring
  -out string
        outputfile
  -query string
        query
```

## Usage parquetreader
```
Usage of ./parquetreader:
  -column string
        column to read
  -filename string
        input to read
```
